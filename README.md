# 비행기 탑승 시뮬레이션

* 간단한 비행기 탑승 시뮬레이션입니다.
* 비행기 탑승은 1명씩 체크인 할 수 있습니다. 다른 한명이 체크인 진행 중이면 체크인을 막아야합니다.
* 우리는 탑승 고객을 순서대로 받아야합니다.(_customersOnBoard)
* 체크인은 출국 심사관에 따라서 랜덤한 시간을 가집니다.
